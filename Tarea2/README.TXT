Realiza una clase para representar a un matemático.
Nuestro matemático es una persona bastante inteligente por lo que es capaz de realizar muchos
cálculos el solo, sin la ayuda de una calculadora ni nadie mas.
Las operaciones que nuestro matemático sabe realizar son:
• Dados tres números: El matemático sabe indicar cual es el numero mayor, cual el menor y
cual del medio.
• Dados la base y altura de un cuadrado: El sabe indicar el área de dicha figura.
• Dados las coordenadas de dos puntos: Sabe calcular distancia entre estos puntos.
• Dado las coordenadas de un punto: Es capaz de indicar en que cuadrante se encuentra dicho
punto.
Tu labor es implementar una clase que nos permite representar al matemático. Debes identificar los
métodos necesarios e implementarlos.
